# Quotify

**Quotify** is a public, API-driven full stack project for managing inspirational quotes and notes.

- Backend: FastAPI (Python) + MongoDB (Atlas)
- Frontend: HTML, CSS, Vanilla JS
- Database: NoSQL (MongoDB Atlas)
